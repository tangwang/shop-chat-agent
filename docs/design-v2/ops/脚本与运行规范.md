# 脚本与运行规范（Python 3.12 / CentOS8）

## 1. 环境准备

```bash
bash scripts/setup_conda_env.sh
source ~/miniconda3/etc/profile.d/conda.sh
conda activate aishopping-py312
```

## 2. 启动脚本约定

- `scripts/start_product_module.sh`
  - 启动商品理解模块（离线摘要服务/查询接口）
- `scripts/start_user_module.sh`
  - 启动用户理解模块（mock画像生成服务）
- `scripts/start_agent_module.sh`
  - 启动导购智能体（orchestrator + assistant）
- `scripts/start_web.sh`
  - 启动 Web（端口 6008）

## 3. 端口

- Web：`6008`
- 其他模块端口建议写入 `.env` 并在脚本中读取。

## 4. 密钥

- `TAVILY_API_KEY`
- `DASHSCOPE_API_KEY`

## 5. 日志目录

建议运行脚本内统一创建：

```bash
mkdir -p Logs
```

并按模块输出：

- `Logs/assistant_YYYYMMDD.log`
- `Logs/orchestrator_YYYYMMDD.log`
- `Logs/product_YYYYMMDD.log`
- `Logs/user_YYYYMMDD.log`
- `Logs/evaluation_YYYYMMDD.log`
